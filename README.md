# Zepto Apps Task

## [Live Application](https://zepto-apps-task.vercel.app)

- Clone the Github repository
```
https://github.com/risalshahed/zepto-apps-task.git
```

- Go inside the project directory
```
cd zepto-apps-task
```

- Install the dependencies
```
npm install
```
- Run the project as follows
```
npm run dev
```

## Features
- Reading Gutendex documentation, each & every data fetched and paginated according to the guideline
- Two Custom hooks used to fetch all of the books and single book with their id
- Pixel Perfect Responsive web application with Vanilla CSS, No UI Templates used
- Data saved in local storage & smooth animations implemented whenever a book item is removed from the wishlist
- Technologies used: Vite, React 18 & React-Router-DOM
